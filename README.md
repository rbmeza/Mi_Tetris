# Mi_Tetris
Para correr el Tetris en local se debe:

1. Crear un entorno virtual (El siguiente comando es para Windows):

```
py -3 -m venv .venv
```

2. Activar el entorno virtual:
```
.venv\Scripts\activate
```

3. Instalar la librer√≠a de pygame:
```
pip install pygame
```

4. Iniciar el juego:
```
python Tetris.py
```
